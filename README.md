# Sistema de Gerenciamento - Estética Automotiva

## Sobre o Projeto

Este é um sistema interno de gerenciamento desenvolvido para estabelecimentos de estética automotiva. A plataforma permite o registro  de clientes (proprietários) e seus veículos, vinculando-os aos serviços.

## Tecnologias Utilizadas

- **Back-end:** Laravel (PHP)
- **Banco de Dados:** SQLite (foco em simplicidade e portabilidade)
- **Front-end:** Blade Templates + Tailwind CSS
